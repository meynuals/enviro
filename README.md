# Enviro3 React Site

Minimal Vite + React (JavaScript) starter.

Quick start (Windows PowerShell):

```powershell
cd "C:\Users\My PC\Documents\enviro3"
npm install
npm run dev
```

Build for production:

```powershell
npm run build
npm run preview
```
